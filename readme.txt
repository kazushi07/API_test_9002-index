■起動の仕方■

①xammpを起動する
②ApacheとMySQLの起動を確認する
③以下のアドレスを開く
http://localhost:9002/index.php

※MySQLでportエラーが出たときは
C:\bin\xampp\mysql\bin
にあるmy.iniのポート番号を書き換える

エラーの確認は
C:\bin\xampp\mysql\data
mysql_error.log

④あるいは、確認したいファイルに沿って移動させる
